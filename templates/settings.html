Runtime Settings
Kiểm tra cấu hình app và deploy
Database mode
TG API credentials
Cần `TG_API_ID` và `TG_API_HASH` để client Telegram hoạt động ổn.
String session
Nên bật để không phụ thuộc file `.session` trên máy local.
Render URL
Khi deploy trên Render, dùng URL này làm đích ping hoặc kiểm tra health.
Deploy Checklist
Checklist tối thiểu để chạy bền hơn
1. `TG_API_ID` + `TG_API_HASH`
2. `TG_STRING_SESSION`
3. `DATABASE_URL` cho Supabase
4. Cron ping `/healthz` Thiết lập ngoài app
App này đã được làm sạch để có thể lên Render như một tool riêng. Bước tiếp theo để chạy bền là chuyển storage sang Supabase và giữ session bằng string thay vì file.
