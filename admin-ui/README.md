# Tele Content Admin UI

This folder is the new MUI skeleton for the app.

## Run

```bash
cd admin-ui
npm install
npm run dev
```

## What it contains

- dark prive_bot-style sidebar
- compact topbar dashboard shell
- metric cards with colored icons
- workspace filter bar + table
- reusable `MuiDialogShell`

## Migration path

Move screens in this order:

1. dashboard
2. workspace table/filter flows
3. settings and dialogs
4. remaining legacy template pages
